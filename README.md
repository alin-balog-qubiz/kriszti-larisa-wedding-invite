# Larisa & Krisztian - Digital Wedding Invitation

A single-page, performant, accessible digital wedding invitation built with plain HTML, CSS, and vanilla JavaScript.

## Project Overview

This is a custom-built wedding invitation for Larisa & Krisztian's wedding.

**Features:**
- Responsive design (mobile-first)
- Bilingual support (Romanian & Hungarian)
- Live countdown timer
- Interactive calendar
- RSVP form

## Getting Started

1. Open `index.html` in a browser
2. Development: Edit files directly or use a local server
